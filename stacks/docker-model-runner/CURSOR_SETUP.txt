Docker Model Runner — Cursor (OpenAI-compatible)

1. Open Cursor Settings (Cmd+,).
2. Go to Models (or search "OpenAI API").
3. Set OpenAI API Key to: not-needed
4. Enable "Override OpenAI Base URL" (wording may vary).
5. Base URL: http://localhost:12434/engines/v1
   (no trailing slash)
6. In the model picker / custom model field, use the full id strings returned by
   GET /engines/v1/models (include docker.io/ai/...:latest). Examples on this host:
   - docker.io/ai/smollm2:latest — small / fast
   - docker.io/ai/llama3.2:latest — general
   - docker.io/ai/qwen3-coder:latest — coding (larger)
   - docker.io/ai/devstral-small-2:latest — agentic coding, 128k context
   Refresh list: curl -s http://localhost:12434/engines/v1/models
   (Also see docker-model-runner/endpoints.json → cursor.modelsFromEnginesV1.)

If something fails: curl http://localhost:12434/engines/v1/models
